Design Philosophy

NoWrong uses a sophisticated dark aesthetic with premium rose accents, elegant typography, and glass morphism effects. The design emphasizes intimacy, privacy, and premium quality through careful use of gradients, shadows, and animations.

Mobile-First Design

Responsive breakpoints and mobile optimization

Mobile (< 768px)

  • 24px padding throughout
  • Single column layouts
  • Full-width buttons
  • Hamburger navigation
  • Touch-optimized 44px targets

Desktop (â‰¥ 768px)

  • Multi-column grids
  • Hover effects enabled
  • Navigation menu visible
  • Larger text sizes
  • Enhanced shadows/glows

Key Breakpoint: 768px

The design uses a single major breakpoint at 768px, transitioning from mobile-first single-column layouts to multi-column desktop layouts. All interactive elements are touch-optimized for mobile while maintaining desktop hover enhancements.

Implementation Notes

Technical details and best practices

Font Loading

Uses Google Fonts Inter with preconnect for performance:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@200;300;400;500;600;700&display=swap">

Performance

  • -webkit-font-smoothing: antialiased
  • will-change: transform on animated elements
  • GPU-accelerated transforms
  • Efficient CSS custom properties

Accessibility

  • Reduced motion support
  • High contrast ratios
  • Semantic HTML structure
  • Focus states on interactive elements

Browser Support

  • backdrop-filter with -webkit- prefix
  • CSS custom properties
  • CSS Grid and Flexbox
  • Modern gradient syntax
